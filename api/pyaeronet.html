pyaeronet¶

pyaeronet.aeronet is an api interface for python. The goal is to make it easy to get AERONET data.

See https://aeronet.gsfc.nasa.gov/print_web_data_help_v3_new.html for more details on the underlying webapi.

Example

import pyaeronet

api = pyaeronet.aeronet()
df = api.to_dataframe(
    site='Cart_Site', SDA20=1, AVG=10,
    year=2000, month=6, day=1,
    year2=2000, month2=6, day2=30,
    add_lst=True
)
ax = df.plot.scatter(x='time_lst', y='Total_AOD_500nm[tau_a]')
ax.figure.savefig('cart.png')
